# Основы Python

**Python** — это высокоуровневый, интерпретируемый, динамически типизированный язык программирования с автоматическим управлением памятью. Он поддерживает объектно-ориентированный, процедурный и функциональный стили программирования.

---

## Типы данных

| Тип данных      | Изменяемость | Хэшируемость | Пример создания                        | Основные методы / операции | Применение | Особенности / дополнительные сведения | Кэширование / оптимизация |
|-----------------|--------------|--------------|----------------------------------------|---------------------------|-----------|--------------------------------------|---------------------------|
| **int**         | ❌ Неизменяемый | ✅ Да        | `x = 10`                                | `+`, `-`, `*`, `/`, `//`, `%`, `**`, `abs()`, `divmod()`, `pow()`, `bit_length()`, `to_bytes()` | Счётчики, индексы, арифметические расчёты, работа с битами | Неизменяемый, arbitrary-precision, поддерживает арифметику, битовые операции, преобразования типов | Малые числа (-5..256) кэшируются, одинаковые объекты повторно используются |
| **float**       | ❌ Неизменяемый | ✅ Да        | `y = 3.14`                              | `+`, `-`, `*`, `/`, `//`, `%`, `**`, `round()`, `is_integer()` | Точные и приблизительные вычисления, измерения, математика | Неизменяемый, поддерживает арифметику с плавающей точкой | Специального кэширования нет, объекты создаются новые |
| **complex**     | ❌ Неизменяемый | ✅ Да        | `z = 1+2j`                              | `.real`, `.imag`, `.conjugate()`, `abs()`, `phase()` (через `cmath`) | Комплексные числа, научные и инженерные расчёты | Неизменяемый, хранит вещественную и мнимую часть | Кэширования нет |
| **bool**        | ❌ Неизменяемый | ✅ Да        | `flag = True`                            | `and`, `or`, `not`, `int()` | Логические операции, условия, флаги | Неизменяемый, наследует от int, два значения: True / False | Кэшируются два объекта: `True` и `False` |
| **str**         | ❌ Неизменяемый | ✅ Да        | `s = "hello"`                            | `upper()`, `lower()`, `strip()`, `split()`, `join()`, `replace()`, `find()`, `count()`, `startswith()`, `endswith()`, `format()` | Хранение текста, обработка строк, вывод информации | Неизменяемый, индексируемый, поддерживает срезы | Короткие строки и идентификаторы часто кэшируются (interning) |
| **bytes**       | ❌ Неизменяемый | ✅ Да        | `b = b"hello"`                           | `decode()`, `hex()`, `startswith()`, `endswith()`, `find()`, `replace()`, `split()`, `join()`, `count()` | Бинарные данные, сети, файлы, кодировки | Неизменяемый, индексируемый, последовательность байт | Кэширования нет |
| **bytearray**   | ✅ Изменяемый   | ❌ Нет       | `ba = bytearray(b"hello")`               | Методы `bytes` + изменение элементов (`ba[0] = 72`) | Изменяемые бинарные данные, буферы, файловые операции | Изменяемый, поддерживает модификацию отдельных байт | Кэширования нет |
| **list**        | ✅ Изменяемый   | ❌ Нет       | `lst = [1, 2, 3]`                        | `append()`, `extend()`, `insert()`, `remove()`, `pop()`, `clear()`, `sort()`, `reverse()`, `count()`, `index()` | Коллекции, последовательности, динамические массивы | Изменяемый, хранит элементы любых типов, индексируемый | Кэширования нет |
| **tuple**       | ❌ Неизменяемый | ✅ Да*       | `t = (1, 2, 3)`                          | `count()`, `index()` | Фиксированные наборы данных, безопасные последовательности, ключи в dict | Неизменяемый, можно хранить разные типы, индексируемый | Короткие неизменяемые кортежи могут кэшироваться |
| **set**         | ✅ Изменяемый   | ❌ Нет       | `s = {1, 2, 3}`                          | `add()`, `remove()`, `discard()`, `pop()`, `clear()`, `union()`, `intersection()`, `difference()`, `symmetric_difference()` | Уникальные элементы, множества, быстрый поиск | Изменяемый, неупорядоченный, уникальные элементы | Кэширования нет |
| **frozenset**   | ❌ Неизменяемый | ✅ Да        | `fs = frozenset([1,2,3])`                | Методы `set`, кроме изменения элементов | Уникальные элементы, ключи dict, математические множества | Неизменяемый, хэшируемый, можно использовать как ключ dict | Кэширования нет |
| **dict**        | ✅ Изменяемый   | ❌ Нет       | `d = {"a":1, "b":2}`                     | `keys()`, `values()`, `items()`, `get()`, `pop()`, `popitem()`, `update()`, `clear()`, `setdefault()` | Ассоциативные массивы, хранение ключ→значение | Изменяемый, ключи должны быть хэшируемыми | Кэширование внутренних структур оптимизировано, но новые объекты dict создаются всегда |
| **NoneType**    | ❌ Неизменяемый | ✅ Да        | `n = None`                               | Проверка: `is None` | Отсутствие значения, инициализация переменных | Неизменяемый, только один объект None | Единственный объект None кэширован |

## Разделы

| Раздел | Ключевые моменты |
|--------|-----------------|
| **Основы языка** | Типы данных, mutable/immutable, хэшируемые типы, `is`, преобразование типов, литералы |
| **Управляющие конструкции** | `if`, `elif`, `else`, тернарный оператор, циклы `for`/`while`, `break`, `continue`, `pass`, генераторы списков и `yield` |
| **Функции** | Определение, lambda, аргументы (`*args`, `**kwargs`), замыкания, функции высшего порядка, области видимости, `return` |
| **Коллекции** | list, tuple, set, frozenset, dict, bytes, bytearray; методы и использование |
| **Исключения** | `try-except-else-finally`, `raise`, пользовательские исключения, типичные ошибки |
| **Модули и пакеты** | `import`, `from ... import ...`, стандартная библиотека, собственные модули, точка входа `if __name__ == "__main__":` |
| **Классы и ООП** | Определение классов, атрибуты и методы, наследование, полиморфизм, специальные методы (`__init__`, `__str__`, `__repr__`), декораторы методов (`@staticmethod`, `@classmethod`, `@property`) |
| **Файлы и потоки** | `open()`, контекстные менеджеры (`with`), методы `read()`, `write()`, `readlines()`, `writelines()`, потоки `sys.stdin`/`stdout`, `io.StringIO`, `io.BytesIO` |
| **Декораторы** | `@decorator`, встроенные декораторы, декораторы с аргументами, кэширование (`functools.lru_cache`), логирование, проверка прав |
| **Контекстные менеджеры** | Методы `__enter__`/`__exit__`, создание через класс или `@contextmanager`, управление ресурсами (файлы, соединения, блокировки) |
| **Особенности Python** | Динамическая типизация, изменяемые vs неизменяемые объекты, кэширование малых чисел и строк, хэшируемые типы для set и dict, `finally`, `is` |
